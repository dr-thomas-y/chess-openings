走到 1.e4 c5 2.Nf3 d6 3.d4 cxd4 4.Nxd4 Nf6 5.Nc3 a6 後，白棋第6步是關鍵分叉點。點擊任一分支展開：
♝
英格利希攻擊
6.Bg5 — 釘住 Nf6
Fischer 最愛，最複雜主線
▶

▸ 白棋計畫

Bg5 釘住黑棋 Nf6，接著 Qd2 + 0-0-0 后翼易位，從 d 列和 h 列猛攻黑棋國王。

▸ 黑棋對應

6...e6 接受釘制，或 7...h6 趕象。接著 Be7 + 0-0 + b5 后翼反攻。

✓ 可以保留

a6 效果完全保留，b5 反攻、Nbd7 + Qc7 配置方向全部不變。

✗ 必須調整

Nf6 被釘住不能輕易移動。需先決定 e6 接受壓制還是 h6 趕象，易位時機謹慎。

♗
英格利希攻擊體系
6.Be3 — 現代最流行
Anand、Carlsen 常用
▶

▸ 白棋計畫

Be3 穩定馬位，接著 f3 → g4 → g5 強力推進王翼，兵鏈直衝黑棋國王。

▸ 黑棋對應

6...e5 趕走 Nd4，或 6...e6 穩健發展。目標是 Nbd7 + b5 + Bb7 后翼賽跑。

✓ 可以保留

后翼計畫完整保留：b5、Qc7、Bb7 一樣，a6 效果不變，e6 + Nbd7 配置相同。

✗ 必須調整

白棋 g4-g5 推進很快，王翼易位後國王可能危險，需判斷是否改為后翼易位。

♗
費雪-索辛變例
6.Bc4 — 瞄準 f7
主教直接威脅 f7 弱點
▶

▸ 白棋計畫

Bc4 直接瞄準 f7，搭配 Nb3 → Nd5 跳馬計畫，Fischer 職業早期招牌。

▸ 黑棋對應

6...e6 封閉主教斜線，接著 Be7 + 0-0 安全易位，也可以 6...e5 趕走 Nd4。

✓ 可以保留

a6 阻止 Bb5+ 效果更重要，后翼反攻大方向保留，b5 和 Qc7 仍是長期目標。

✗ 必須調整

不宜急於易位，f7 有戰術威脅，e6 的時機要比其他分支更早走。

♙
六兵攻擊
6.f4 — 激進王翼推進
最激進，賽跑感最強
▶

▸ 白棋計畫

f4 宣示王翼意圖，接著 f5 → g4 → g5，強力兵鏈衝向黑棋國王，節奏最快。

▸ 黑棋對應

6...e5 立即反中央趕走 Nd4，或 6...Qc7 控制中路，必須快速製造后翼壓力。

✓ 可以保留

b5 + d5 反攻的大方向完全保留，a6 依然有效，Nbd7 配置可以保留。

✗ 必須調整

節奏必須加快，白棋 e5 是真實威脅，后翼反攻要比其他分支更急迫。

✦ 納依道夫的不變法則

1
a6 永遠有效：不管白棋走什麼，只要沒有立即戰術威脅，a6 都是正確的。它封鎖 b5，是納依道夫的生命線。
2
后翼反攻優先：黑棋的目標永遠是 b5 推進和 d5 反擊。不管白棋在王翼怎麼動，黑棋都要製造后翼反壓。
3
Nf6 是核心棋子：這匹馬控制 d5 和 e4，白棋最想釘住它，黑棋要保護或主動交換。
4
d6 決定主教方向：d6 的兵決定 c8 主教走 e6 或 d7，是整個子力配置的關鍵判斷點。

卡羅康防禦 主線變例

走到 1.e4 c6 2.d4 d5 3.Nc3 dxe4 4.Nxe4 之後，白棋有四種不同方向，黑棋的應對各不相同：
♞
古典變例（主線）
4...Nd7 — 穩健發展
Karpov、Petrosian 最愛
▶

▸ 白棋計畫

5.Nf3 + Bc4/Bd3 發展，王翼易位，利用先手優勢製造中央壓力。白棋兵形健全，優勢穩健。

▸ 黑棋對應

4...Nd7 支撐 e6，接著 Ngf6 + Bd6 穩固發展。c8 主教可以走 b7 或 g4，是卡羅康最穩健的路線。

✓ 可以保留

c6 阻止白棋在 d5 建立強大兵形的效果完整保留，e6 支撐 d5 的結構思路不變。

✗ 必須調整

c8 主教的發展路線需要提前規劃，不要讓它被 e6 的兵永久封住。Nd7 雖穩健但有暫時遮擋主教的問題。

♞
現代變例
4...Nf6 — 積極對攻
Bf5 搶先出動主教
▶

▸ 白棋計畫

5.Nxf6+ gxf6（或 exf6），攻擊黑棋被破壞的兵形。白棋的目標是利用 f 列半開放線和黑棋的雙倍兵弱點。

▸ 黑棋對應

4...Nf6，如果 5.Nxf6+ 接受換馬，黑棋走 gxf6 雖然兵形破壞，但換來了兩個主教和半開放 g 列，可以積極反攻。

✓ 可以保留

c6 + d5 的中央骨架思路保留，黑棋的整體目標仍是穩固後發展雙主教優勢。

✗ 必須調整

接受 Nxf6+ 後兵形變化較大，需要接受 g 列雙倍兵，換取主動反攻機會，心態上是「損形換活力」。

♙
前進變例
3.e5 — 法式風格
白棋鎖住中央，空間優勢
▶

▸ 白棋計畫

3.e5 鎖住中央，取得空間優勢，接著 f4 鞏固 e5 陣地，後續 Nf3 + Bd3 + 0-0 進行穩健發展。

▸ 黑棋對應

3...Bf5 是最重要的一步：搶在 e6 關門之前把 c8 主教出來。接著 e6 + Nd7 + Ne7 + Nc6，準備用后翼兵推反攻白棋的 e5 陣地。

✓ 可以保留

后翼兵推（c5 或 f6）反攻 e5 的整體方向保留，只要 Bf5 及時出動，主教活力就不會喪失。

✗ 必須調整

比主線更需要提前規劃 c5 的時機。白棋的空間優勢很明顯，黑棋必須積極反攻，不能被動等待。

♗
兩馬變例
2.Nc3 — 快速發展
避開 d4 正面衝突
▶

▸ 白棋計畫

2.Nc3 繞過 2.d4，快速發展子力，接著 d4 + Nf3 + Bc4，製造比主線更靈活的攻擊。

▸ 黑棋對應

黑棋同樣走 2...d5，接著 3.Nf3 Bg4！（釘住白棋的 Nf3）或 3...dxe4 進入主線形式。維持卡羅康的穩健結構。

✓ 可以保留

d5 控制中央的思路完整保留，卡羅康的核心邏輯（穩固 d5，避免法式主教問題）在此分支完全適用。

✗ 必須調整

白棋的開局更靈活，黑棋不能按固定順序走，需要根據白棋選擇決定先 Bg4 釘馬還是先進行換子。

✦ 卡羅康的不變法則

1
c6 + d5 是骨架：先走 c6，再走 d5。c6 讓 d5 的兵有後盾支撐，不需要用后來保護，這是卡羅康與法式的最大差異。
2
提早出動 c8 主教：卡羅康最大優點就是 c8 主教不會像法式那樣被鎖死。遇到前進變例時，Bf5 是最高優先的一步。
3
穩健不代表被動：卡羅康的目的是建立健全兵形後反攻，Karpov 用它打出許多精彩的后翼反壓。穩固是手段，不是目的。
4
兵形完整是最大優點：相較於西西里，卡羅康黑棋的兵形幾乎沒有弱點。在殘局中這個優勢非常顯著。
白棋變例黑棋核心應對關鍵優先步核心原則
主線 3.Nc33...dxe4 4...Nd7 或 Nf6Bf5 或 Bg4穩健發展，保留雙主教
前進 3.e5立即 3...Bf5Bf5 絕對優先主教出來前 e6 不能走
交換 3.exd53...cxd5 對稱Nf6 + Bf5接受對稱，謀求殘局
兩馬 2.Nc32...d5，靈活應對視情況 Bg4 或換子保持 d5 控制，核心不變

后翼棄兵 接受變例

Queen's Gambit Accepted — QGA  |  執白

—
走到 1.d4 d5 2.c4 dxc4 3.Nf3 Nf6 4.e3 或 4.e4 後，白棋有不同的中央計畫，目標都是吃回 c4 的兵並建立中央優勢：
♗
古典路線
3.e3 + Bxc4 — 穩健吃回
最直接，快速建立中央
▶

▸ 白棋計畫

3.e3 準備 Bxc4 吃回兵，接著 Nc3 + 0-0，建立 d4+e3 的穩健中央。Bc4 對準 f7，威脅性強。

▸ 黑棋對應

黑棋通常走 a6 + b5 保住 c4 的兵，或者 e6 + c5 反擊白棋的 d4，製造不對稱局面。

✓ 白棋優勢

發展迅速，Bc4 + Nc3 + 0-0 的配置非常自然流暢，對 f7 的壓力是長期威脅。

✗ 注意事項

如果黑棋走 a6 + b5 保住兵，白棋不要急著強行奪回，而是繼續發展加速節奏壓制。

♘
中央強攻路線
3.e4 — 中央雙兵
Kasparov 最愛，最激進
▶

▸ 白棋計畫

3.e4 直接建立 d4+e4 的中央雙兵，形成超強的中央控制。接著 Bxc4、Nc3 + 0-0，王翼進攻。Kasparov 在多場大賽中用此路線制勝。

▸ 黑棋對應

黑棋可以走 3...e5 立即反擊中央，或 3...Nf6 先發展再等時機。如果讓白棋維持雙兵，黑棋會非常被動。

✓ 白棋優勢

中央控制極強，王翼攻勢後勁十足。d4+e4 的雙兵是白棋的理想型態，空間優勢顯著。

✗ 注意事項

雙兵中央需要穩定支撐，如果黑棋成功用 e5 或 c5 打破中央，白棋的兵形可能變成弱點。

♜
拒絕棄兵
2...e6 — 后翼棄兵拒絕
黑棋拒絕接受，轉入 QGD
▶

▸ 白棋計畫

如果黑棋拒絕接受（走 e6 而非 dxc4），轉入 后翼棄兵拒絕（QGD）。白棋接著 Nc3 + Nf3 + Bg5，加壓黑棋的中央。

▸ 黑棋對應

黑棋走 e6 保住 d5 的兵，接著 Nf6 + Be7 + 0-0，進入穩健的防禦體系。黑棋接受略微被動的局面換取兵形健全。

✓ 白棋優勢

空間優勢保持，Bg5 + Qc2 的組合可以製造對 e7 的釘住壓力，中長期優勢明顯。

✗ 注意事項

QGD 的防禦體系非常嚴密，白棋不能急於求勝，需要耐心等待黑棋出錯或在殘局中兌換出優勢。

♙
斯拉夫防禦
2...c6 — 斯拉夫
黑棋保留 c8 主教
▶

▸ 白棋計畫

黑棋走 c6 支撐 d5，不讓 c8 主教被封住。白棋接著 3.Nf3 Nf6 4.Nc3 dxc4，進入斯拉夫主線，繼續爭奪 c4 的兵。

▸ 黑棋對應

斯拉夫的優點是保留了 c8 主教的活動空間（不像 e6 會把主教封死）。黑棋接著 Bf5 或 Bg4 主動出動主教。

✓ 白棋計畫

白棋的目標是 Bg5 釘住 Nf6，或 e4 強行打開中央。Meran 變例（e3+b4）是最精彩的進攻路線。

✗ 注意事項

斯拉夫的理論很深，黑棋防禦工具完整。白棋需要比 QGD 更精準的計算才能獲得優勢。

✦ 后翼棄兵執白的不變法則

1
c4 是棄兵，不是失誤：讓黑棋吃掉 c4 是計畫的一部分。白棋的目的是用這個兵換取快速發展、中央控制和開放的主教斜線。
2
d4 是白棋的生命線：整個開局的邏輯都圍繞 d4 的兵。一旦 d4 的兵被打掉，白棋的優勢就大幅縮水。保護 d4 是最高優先。
3
Bxc4 比 d4 進一步重要：白棋要先吃回 c4 的兵讓主教激活，再考慮其他計畫。Bc4 在對角線上的壓力是白棋的核心武器。
4
節奏就是優勢：黑棋多走了一步吃兵（dxc4），白棋的每一步都應該是有效的發展，不能浪費。Nf3 + Nc3 + 0-0 是最高效的發展順序。

2...c6（斯拉夫）爭奪 c4，主教斜線競賽e3 + Bd3如何在兩翼同時施壓？

倫敦體系 標準建立

London System — White Opening  |  執白
← → 鍵盤翻頁

建立理想陣型時，對手可能挑戰你的主教陣型。點擊展開學習應對：
♝
對手走 ...Bd6
攻擊你的 Bf4
不要換！退主教回 Bg3
▶

▸ 對手的意圖

對手用 ...Bd6 直接攻擊你的 Bf4，希望你換主教，失去對 e5 格的控制。

▸ 你的正確應對

不要換主教！把 Bf4 退回 Bg3。如果對手換了（Bxg3），你用 hxg3 重吃，開放 h 縱線給車！

✓ 退主教的好處

hxg3 開放 h 縱線，車發動王翼攻擊。e5 控制完整保留，Ne5+Qh5 攻擊照常執行。

✗ 千萬不要

不要直接 Bxd6 換主教！換了之後對手重吃，e5 格失去控制，Ne5 攻擊就此瓦解。

♝
對手走 ...Bf5
搶佔斜線控制
立刻 Bg3 挑戰，讓他先換
▶

▸ 對手的意圖

對手走 ...Bf5，佔領斜線，限制你的子力空間，準備換掉你的主教。

▸ 你的正確應對

立刻走 Bg3 挑戰，但不要主動換主教。讓對手先換（Bxg3），你再用 hxg3 重吃，開放 h 縱線！

✓ 策略要點

讓對手做換子決定。若對手不換，Bg3 繼續控制局面。若換了，h 縱線開放是攻擊獎勵。

✗ 千萬不要

不要主動走 Bxf5 換主教。你要保持主動，讓對手決定是否換子。

♟
對手推 ...c4
攻擊你的 Bd3
退 Bc2，主教仍指向 h7
▶

▸ 對手的意圖

對手推 ...c4 攻擊 Bd3，希望趕走主教並佔領后翼空間。

▸ 你的正確應對

把 Bd3 退回 Bc2（退一格）。主教在 c2 斜線上依然指向 h7，攻擊力完全保留！

✓ 退主教的好處

Bc2 繼續瞄準 h7，Ne5+Qh5 攻擊一步都未受影響。對手后翼佔地，你王翼攻殺。

✗ 別被后翼分心

不要跟著去后翼防守。倫敦體系的王翼攻擊通常比對手的后翼反壓更快、更致命。

✦ 倫敦體系的不變法則

1
Bf4 必須在 e3 之前！：這是最重要的順序。先走 Bf4，主教才能出來。若先走 e3，c1 主教被自己的兵永久封死，整個陣型瓦解。
2
e5 格是你的命根子：兵陣（c3-d4-e3）、Bf4 和 Nf3 的任務都是控制 e5。Ne5 是你的主要武器，所有佈局都為它服務。
3
不主動換主教：遇到 ...Bd6 或 ...Bf5，先退主教（Bg3），讓對手決定是否換。換了之後 hxg3 開放 h 縱線，是額外的攻擊資源。
4
Ne5 → Qh5 是主套路：陣型完成後跳 Ne5。對手吃馬，兵重吃，騎士退，立刻 Qh5 壓王翼。威脅 h7 將軍，配合 h4-h5 推進攻殺。
